# Diskusi Interaktif
Aplikasi diskusi tanya-jawab berbasis web untuk pembelajaran matematika.

## Fitur
- Dukungan notasi LaTeX (MathJax)
- Login dengan Google (Firebase Auth)
- Menyimpan pertanyaan/jawaban ke Firestore
- Upload gambar ke Firebase Storage

## Instalasi
Host via GitHub Pages atau jalankan secara lokal dengan Firebase konfigurasi.
